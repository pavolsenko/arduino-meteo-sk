# 04 Zapojenie modulu na meranie teploty

```C
String sensor() {
    float h = dht.readHumidity();
    float t = dht.readTemperature();
    String result;

    while(isnan(h)) {
        h = dht.readHumidity();
    }

    while(isnan(t)) {
        t = dht.readTemperature();
    }

    result = String(t, 0) + "'C / ";
    result += String(h, 0) + "%";

    return result;
}
```
